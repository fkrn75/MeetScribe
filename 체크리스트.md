# MeetScribe 개발 체크리스트 (Checklist)

> **작성일**: 2026-06-17 · **상태**: 제작 전 (착수 시 체크 시작)
> 마일스톤 순서대로 진행. 각 M은 "다음으로 넘어가도 되는" 완료 기준(DoD)을 가짐.

---

## M0. 환경 셋업
- [ ] Python 3.10+ 가상환경 구성
- [ ] `pip install whisperx faster-whisper pyannote.audio fastapi uvicorn`
- [ ] ffmpeg 설치·경로 확인
- [ ] Hugging Face 계정 + **pyannote 3.1 사용 동의** + 토큰 발급
- [ ] (GPU 사용 시) CUDA 대응 torch 설치 확인 (`torch.cuda.is_available()`)
- [ ] Rust + Node + Tauri 2 CLI 설치 (`cargo tauri --version`)
- **DoD**: `python -c "import whisperx, torch; print(torch.cuda.is_available())"` 정상

## M1. 백엔드 코어 — CLI로 먼저 검증 ⭐ (제작의 첫 실증)
- [ ] `preprocess.py`: ffmpeg로 임의 파일 → 16kHz mono WAV
- [ ] `transcribe.py`: faster-whisper 한국어 전사 (VAD, 환각 억제 옵션)
- [ ] `align.py`: 한국어 wav2vec2 단어 정렬
- [ ] `diarize.py`: pyannote 화자 분리
- [ ] `merge.py`: 화자↔단어 병합 → 세그먼트 JSON
- [ ] CLI 한 줄로 `파일 → 화자별 JSON/SRT` 출력
- [ ] 실제 한국어 회의 샘플로 품질 육안 검증
- **DoD**: 샘플 회의 파일 → 화자별 SRT가 합리적으로 생성됨

## M2. API 서버화 + 사이드카 연결
- [ ] `app.py`: FastAPI 라우팅(`/health`, `/jobs`, `/system`)
- [ ] `jobs.py`: 작업 큐 + 백그라운드 스레드(비차단)
- [ ] `events.py`: SSE 진행률 스트림(단계 가중치)
- [ ] `/jobs/{id}/cancel` 취소 동작
- [ ] PyInstaller로 사이드카 바이너리 빌드(타겟 트리플 네이밍)
- [ ] Tauri `externalBin` 등록 + spawn/health/kill lifecycle
- **DoD**: Tauri가 사이드카 기동 → `/health` OK → 작업 요청·진행률 수신

## M3. 최소 GUI
